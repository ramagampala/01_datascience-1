K-Nearest Neighbours (An instance-based learning or lazy learning)¶

  • Unlike model based learning, instance-based learning will store training data and pospone generalization until a new instance is required to be classified or predicted.
  • KNN can be used for both classification and regression predictive problems. However, it is more widely used in classification problems
  • Below picture depicts how KNN algorithm works.
  • The distance between the new instance and the records is calculated using euclidean distance $\sqrt{(x_1 - p_1)^2 + (x_2 - p_2)^2 + ... + (x_n - p_n)^2}$
  • It is important to normalize the data, as KNN looks at the distances. If data is scalled, the distances will be appropriate.
  • Use learning curve to choose K.

Error Analysis and choosing K¶

  • One of the ways to evaluate Mulit-Label classifier is to calulate F1-Score for each label, and then average all these (f1_score with average = 'macro'). This is not be best approach as it will not consider the label weights (number of rows with label > 5 vs number of odd labels) and will not account for label imbalance (this problem can be addressed with f1_score with average = 'weighted').

def plot_learning_curves(model, X, y):
    X_train, X_val, y_train, y_val = train_test_split(X, y, test_size=0.2, random_state=10)
    train_error, val_error, sample_size = [], [], []
    for m in range(11, len(X_train)):
        model.fit(X_train[:m], y_train[:m])
        y_train_predict = model.predict(X_train[:m])
        y_val_predict = model.predict(X_val)
        
        train_f = f1_score(y_train_predict, y_train[:m], average="weighted")
        train_error.append(1 - train_f)
        
        val_f = f1_score(y_val_predict, y_val, average="weighted") 
        val_error.append(1 - val_f)
        
        sample_size.append(m)

    plt.plot(sample_size, train_error, "r-+", linewidth=2, label="train")
    plt.plot(sample_size, val_error, "b-", linewidth=3, label="val")
    plt.legend(loc="bottom right", fontsize=14)   
    plt.xlabel("Training set size", fontsize=14) 
    plt.ylabel("F1-Score", fontsize=14)
