Gradient Descent Convergence



i) Min-Max Scaling (normalization) : Values are shifted and re-scaled so that they end up ranging between 0 and 1. We do this by subtracting MIN value and divide by MAX minus MIN value. Scikit-Learn provides a transformer called MinMaxScaler, there is a hyper parameter feature_range that lets you change the range to some thing other than 0 and 1.

 

ii) Standardization : Subtract MEAN value and then divide by STANDARD DEVIATION (standardized values always have 0 mean), so that the resulting distribution has a unit variance.
